# Aggiornamento v3

1. Eseguire `add_new_products_and_languages.sql` in Supabase SQL Editor.
2. Caricare su GitHub almeno `admin.js` aggiornato.
3. Le 8 nuove focacce vengono create come bozze con prezzo 0,00 e senza badge.
4. Le nuove lingue sono disponibili nei campi descrizione dell'admin:
   PL, SQ, RO, HU, NL, HR, EL, TR.
5. Il nome della focaccia resta unico in italiano per tutte le lingue.
