# Menu Multilingua Test

File principali:
- `index.html`: selezione lingua, per ora solo italiano
- `menu_IT.html`: menu pubblico dinamico con grafica originale
- `admin.html`: area privata di gestione
- `config.js`: collegamento al progetto Supabase

## Accesso admin
Aprire `/admin.html` e usare l’utente creato in Supabase Authentication.

## Pubblicazione
Caricare tutti i file nella root del repository e attivare GitHub Pages dal branch `main`, cartella `/root`.
